# Netflix Content Analysis Dashboard  
### Python + Power BI End-to-End Data Project

---

## 📌 Project Overview

This project analyzes Netflix's content dataset by performing data cleaning and preprocessing in Python, followed by interactive dashboard development in Power BI.

The objective was to transform an uncleaned raw Excel dataset into a structured, insight-driven dashboard using modern data analysis tools.

---

## 🧹 Data Cleaning & Preparation (Python)

The original Excel dataset contained:
- Missing values
- Inconsistent country names
- Text-based duration fields
- Null and "Unknown" entries

### 🔧 Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### 📊 Cleaning Steps Performed:
- Handled missing values
- Removed "Unknown" country entries
- Extracted numeric duration from text
- Standardized column names
- Created structured duration column
- Exported cleaned dataset for BI analysis

---

## 📊 Power BI Dashboard Development

After cleaning the dataset in Python, the structured data was imported into Power BI to create an interactive analytical dashboard.

### 🛠 Tools Used:
- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling

---

## 📈 Key KPIs

- 🎥 Total Titles: 7,881
- 🎬 Total Movies: 5,689
- 📺 Total TV Shows: 2,192
- ⏱️ Average Movie Duration: 104 mins

---

## 📊 Dashboard Insights

- Movies account for approximately 73% of total content.
- Significant growth in content addition after 2015.
- United States leads in content production.
- TV-MA and TV-14 are the most common ratings.
- Average movie duration is approximately 104 minutes.
